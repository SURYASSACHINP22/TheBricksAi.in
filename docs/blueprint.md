# **App Name**: BrickAi User App

## Core Features:

- User Authentication: Enable users to securely sign up, log in, and manage their profiles, including name, phone number, and owned properties.
- Property Listing: Allow users to list properties for sale by entering details such as survey number, location, dimensions, and uploading relevant documents. Generates a unique Property ID and sets the initial verification status to USER_DECLARED.
- Buy Property: Enable users to browse and search for properties, view property details (including land records, location, and documents), and filter by type, location, and price. Display properties from BrickAi projects, verified developers, and individual owners with corresponding source labels.
- Nearby Amenities: Display nearby amenities (hospitals, schools, transport, shops) on the property details page, including name, category, distance, travel time, rating, and address, sorted by distance, along with a visual distance graph.
- AI House Planning: Generate conceptual house plans based on user inputs such as property details, number of floors and rooms, budget, and preferences. Generates basic layouts, room distributions and orientation notes, along with a placeholder 3D image. LLM is used as a tool to include important building constraints into consideration when planning layouts.
- Plan Improvement: Allow users to upload external house plans and request AI improvements. Display before-and-after views of the plan with explanations of the changes.
- Price & Market Information: Display historical price trends and current price ranges for properties, along with a disclaimer about the indicative nature of the prices.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to convey trust and transparency in property information.
- Background color: Very light blue (#F0F8FF), almost white, to ensure a clean and trustworthy backdrop for the content.
- Accent color: Soft green (#98FB98) for CTAs and highlighting important data, symbolizing growth and positive investment.
- Body font: 'Inter' (sans-serif) for clear and accessible information.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and technical look.
- Use clear and simple icons to represent property types, amenities, and actions.
- Maintain a clean and structured layout, prioritizing property data and land record details above all.
